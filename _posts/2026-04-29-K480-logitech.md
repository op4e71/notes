title: Logitech K480 permanent function keys
---

Logitech K480 comes with function keys that are available as secondary function. 
To make function keys primary you can use Logitech tool for Windows. For my keyboard I found that it works on 
Windows profile (keyboard can have 3 profiles configured that are switched with the rotary switch). 
The setting on one profile is not populated to the other profile. For me solaar tool (on Linux) did not work (the keyboard was not detected). 
There is also a small tool [link](https://github.com/ericmbf/k480-ubuntu-fn-invert-auto), though it does not recognize my version of the keyboard.


Reconfiguration of the keyboard is done with the byte sequence. The default sequence `{0x10, 0xff, 0x0b, 0x1e, 0x00, 0x00, 0x00}` does not work on my keyboard.

To make my keyboard work with function keys working as primary:
- Change ID to `0xb330` (the ID of my K480 version)
- Change the sequence to:  `{0x10, 0xff, 0x07, 0x15, 0x01, 0x00, 0x00}`
